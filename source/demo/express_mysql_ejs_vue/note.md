# NOTE

## 2017-06-30
1. 项目初始化
```sh
npm init
```
2. 添加第三方包
```sh
npm install express --save
npm install mysql --save
npm install ejs --save
npm install body-parser --save
```
3. 创建一个文件或目录
```sh
touch note.md
touch app.js
touch lib/Config.json
mkdir -p lib/demo
mkdir -p lib/utils
mkdir public
mkdir public/js
mkdir public/css
mkdir public/plugins
```
