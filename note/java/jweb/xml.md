# XML
Extensible Markup Language 可扩展标记语言, w3c组织发布，目前为2000年的1.0规范

除了用来保存有关的数据以外，还经常用作配置文件，描述模块之间的关系

## 语法
* 文档声明
* 元素
* 属性
* 诸事
* CDATA区、特殊字符
* 处理指令

## 文档声明
```xml
<!-- 最简单的申明 -->
<?xml version="1.0" ?>
<!-- encoding属性说明文档编码 -->
<?xml version="1.0" encoding="UTF-8" ?>
<!-- standalone属性说明文档是否独立 -->
<?xml version="1.0" encoding="UTF-8" standalone="yes" ?>
```
